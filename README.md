📚 Sistema de Registro de Alunos
Aplicação desktop completa para registro de alunos desenvolvida em Python, com interface gráfica moderna e intuitiva utilizando Tkinter e CustomTkinter.

✨ Funcionalidades
- Cadastro de novos alunos com dados pessoais e acadêmicos

- Busca, edição e exclusão de registros de alunos

- Armazenamento persistente dos dados (SQLite / JSON / CSV)

- Interface moderna e personalizável com CustomTkinter

- Exibição organizada dos alunos cadastrados em formato de lista/tabela

- Filtros e ordenação por nome, matrícula ou curso

🛠️ Tecnologias Utilizadas
- Python 3

- Tkinter – biblioteca padrão para interfaces gráficas

- CustomTkinter – componentes modernos e personalizáveis para UI

- SQLite3 (banco/sistema de arquivos para armazenamento)

🚀 Como Executar

git clone https://github.com/YuriB4rbosa/sistema-registro-alunos.git
cd sistema-alunos
pip install -r requirements.txt


# Caso queira ele como .exe 
pip install pyinstaller
python -m PyInstaller SistemaAcademico.spec
